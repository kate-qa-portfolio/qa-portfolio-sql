# 🗄 Oracle HR SQL Portfolio Project
## 📌 Description
SQL-проєкт на основі Oracle HR database з практичними запитами для роботи з даними.
Проєкт включає:
- SELECT, WHERE, ORDER BY
- JOIN (об’єднання таблиць)
- GROUP BY, HAVING (агрегація)
- Subqueries (підзапити)
- CASE expressions
- UNION / UNION ALL
- CRUD операції (INSERT, UPDATE, DELETE)
## ⚙️ Database
Oracle HR sample database
## 🛠 Skills
SQL (Oracle), data filtering, joins, aggregation, subqueries, data manipulation (CRUD)
## 🎯 Purpose
Практика SQL навичок для позиції Junior QA Engineer (робота з базами даних та перевірка даних).
