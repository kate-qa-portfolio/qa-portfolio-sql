# 🗄 Oracle HR SQL Portfolio Project

## 📌 Description
SQL-проєкт портфоліо на основі бази даних Oracle HR.  
Містить практичні запити для витягування та перевірки даних, агрегації, об’єднань таблиць (JOIN), підзапитів та CRUD-операцій.

---

## 🧠 What this project includes

### 🔹 SELECT & Filtering
- Вибір даних з таблиці Employees
- Умови WHERE (salary, department, commission)
- Робота з LIKE, IN, BETWEEN

### 🔹 Aggregation
- COUNT()
- GROUP BY
- HAVING
- Сортування результатів (ORDER BY)

### 🔹 JOIN operations
- INNER JOIN між таблицями:
  - LOCATIONS
  - COUNTRIES
  - REGIONS

### 🔹 Subqueries
- Фільтрація даних через вкладені запити
- Аналіз працівників по департаментах та історії роботи

### 🔹 CASE statement
- Перетворення LOCATION_ID у назви штатів

### 🔹 UNION / UNION ALL
- Об’єднання даних з EMPLOYEES та JOB_HISTORY
- Формування статусів працівників

### 🔹 CRUD operations
- INSERT нового працівника
- UPDATE існуючого запису
- DELETE тестового запису
